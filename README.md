## Installation

* Download and install [Neard](https://github.com/crazy-max/neard).
* Download one of the available PHP bundles.
* Extract archive in `neard\bin\php\`.

Directory structure :
```
[-] neard
 | [-] bin
 |  | [-] php
 |  |  | [-] php5.3.13
 |  |     | neard.conf
 |  |  | [-] php5.6.14
 |  |     | neard.conf
 ```
